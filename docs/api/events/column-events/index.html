列事件

columnsChanged#

() => void;

网格列被重置时触发，修改列选项时不会被触发。

columnHeightChanged#

(height?: number) => void;

列的高度被修改时触发。

参数类型描述
heightnumber高度

columnPinnedChanged#

(column?: string, pinned?: ColumnPinned) => void;

列的固定状态被修改时触发。

参数类型描述
columnstring当前被修改的列 field
pinnedColumnPinned修改后的固定状态

columnVisibleChanged#

(column?: string, visible?: boolean) => void;

列的显示状态被修改时触发。

参数类型描述
columnstring当前被修改的列 field
visibleboolean修改后的列显示状态

columnWidthChanged#

(column?: string, width?: number) => void;

列的宽度被修改时触发。

参数类型描述
columnstring当前被修改的列 field
visiblenumber修改后的列宽度

columnNameChanged#

(column?: string, name?: string) => void;

列的名字被修改时触发。

参数类型描述
columnstring当前被修改的列 field
namestring修改后的列名称

groupCollapsedChanged#

(group?: string, collapsed?: boolean) => void;

列分组折叠状态被修改时触发。

参数类型描述
groupstring当前被修改的分组 id
collapsedboolean修改后的分组折叠状态

groupNameChanged#

(group?: string, name?: string) => void;

列分组名称被修改时触发。

参数类型描述
groupstring当前被修改的分组 id
namestring修改后的分组名称
